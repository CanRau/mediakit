# @media-kit/solid

A set of utilities to use with your Solid apps.

### Installation

```bash
pnpm install @solid-mediakit/core
```

### Usage

```tsx
import { Toggler } from '@solid-mediakit/core'

const Home = () => {
  return (
    <div>
      <Toggler />
    </div>
  )
}

export default Home
```
